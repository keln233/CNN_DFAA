The source code used for CNN model training, evaluation, and prediction is publicly available at https://github.com/keln233/CNN_DFAA. The data availability statement and example workflow are provided at https://github.com/keln233/CNN_DFAA/blob/main/DATA_AVAILABILITY.md.

The meteorological data used in this study were obtained from the China Meteorological Administration. Because of file-size limitations and data redistribution restrictions, the complete raw datasets are not stored in the GitHub repository. The original data sources and processing workflow are described in the repository. The processed basin-level feature tables and trained CNN weights are intermediate analysis products and can be deposited in a public data repository with a DOI if required by the journal or reviewers.
